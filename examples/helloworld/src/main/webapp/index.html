HelloWorld API

Simple HelloWorld API built with Jakarta Servlets

Available Endpoints

GET /api/hello

Returns a JSON hello world message with timestamp

Example Usage

Get Hello Message:

curl http://localhost:9292/helloworld/api/hello

Example Response:

{
  "message": "Hello World from HelloWorld API!",
  "timestamp": "2025-09-02T17:30:00",
  "service": "helloworld",
  "version": "1.0.0",
  "status": "success"
}

Features

  • ✅ Simple JSON response
  • ✅ CORS enabled for development
  • ✅ Timestamp included
  • ✅ No database dependencies
  • ✅ Ready to use
